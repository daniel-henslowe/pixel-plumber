# Pixel Plumber

A modern NES-style platformer built with vanilla JavaScript and HTML5 Canvas.

## Play Now

[**Play Pixel Plumber**](https://daniel-henslowe.github.io/pixel-plumber/)

## Features

- Classic side-scrolling platformer gameplay
- Procedurally generated levels across 4 worlds
- NES-inspired pixel art graphics
- Retro sound effects using Web Audio API
- Power-ups (mushrooms make you big!)
- Multiple enemy types (Goombas and Koopas)
- Coins, bricks, and question blocks
- Mobile touch controls
- Responsive design

## Controls

### Keyboard
- **Arrow Keys / WASD** - Move
- **Space / W / Up** - Jump
- **Shift** - Run

### Mobile
- **D-Pad** - Move
- **A Button** - Jump
- **B Button** - Run

## Technical Details

Built with:
- Pure JavaScript (no frameworks)
- HTML5 Canvas for rendering
- Web Audio API for sound effects
- CSS for UI overlay

No build step required - runs directly in the browser.

## Local Development

1. Clone the repository
2. Open `index.html` in a browser
3. Or use a local server: `npx serve .`
